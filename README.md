**Platform** : macOS (Intel & Apple Silicon) Only

Based on the [Nayuki QR Code generator](https://github.com/nayuki/QR-Code-generator) (MIT License)

## Description
Simple Jitter external to generate QR codes directly as matrices in Max.

This repository hosts binary releases of the external.

![](https://github.com/gluon/jit.void.qrcode/blob/main/StructureVoid.png)

## Installation
Unzip and copy the `jit.void.qrcode` folder into:
`~/Documents/Max 9/Packages`
